# NestJS snippets

Type-checked TypeScript snippets referenced from [`docs/integrations/nestjs.md`](../../docs/integrations/nestjs.md).

These snippets are **not a runnable app**. They exist so the documentation stays compilable as `@junando/core`, NestJS, or `nestjs-pino` evolve. CI runs `pnpm --filter @junando/nestjs-snippets exec tsc --noEmit` and fails if any snippet stops type-checking.

If you change something here, update the corresponding section in the guide.
